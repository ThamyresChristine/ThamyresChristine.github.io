História da Web

(Voltar)

É impossível falar da Internet sem falar da principal ferramenta que utilizamos para acessá-la: o computador. Então vamos lá! De acordo com o dicionário Michaelis, A palavra computador significa "aquele ou aquilo que calcula baseado em valores digitais; calculador, calculista". Computadores nasceram da necessidade do homem de realizar cálculos completos em um curto período. O Ábaco, provavelmente criado na Babilônia, é considerado como o ancestral do computador.

A construção do computador que conhecemos hoje deve-se especialmente ao avanço da Matemática, da Física, a 2ª Guerra Mundial e a “Guerra Fria”. Vamos conhecer cada uma de suas fases:

  • 1ª geração (1940-1956): Uso de válvulas como componentes básicos da memória, o que consumia muita energia. Criação do ENIAC.
  • 2ª geração (1957-63): Época dos transistores, houve uma melhoria no desempenho dos computadores e um menor gasto de energia. Criação das linguagens de programação COBOL, FORTRAN e Assembly.
  • 3ª geração (1965-70): É a vez dos circuitos integrados na construção dos computadores, dando uma nova melhoria no desempenho das máquinas não somente nas questões computacional e consumo de energia, mas também nos hardwares. Foram lançadas novas versões do COBOL e FORTRAN e houve a criação da linguagem Pascal.
  • 4ª geração (décadas de 1970-80): Com a utilização dos circuitos VLSI (Very Large Scale Integrated) combinando milhares de transistores em um único chip – os processadores, os computadores se tornaram mais compactos, rápidos, poderosos na questão do processamento e com um custo acessível. Foram criadas as linguagens de programação C e C++.
  • 5ª geração (Anos 80 até os dias atuais): Substituição dos circuitos VLSI para os circuitos ULSI (Ultra Large Scale Integrated) também conhecidos como os microchips, permitiu a comercialização dos PC’s – Personal Computers ou computadores pessoais. Surgimento da Microsoft com o sistema operacional Windows. Nos últimos anos, tivemos o advento da IA – Inteligência Artificial sendo integradas aos softwares.

Grandes Nomes

  1. Ada Lovelace: considerada a primeira programadora da história, ela ajudou Charles Babbage ao desenvolver um algoritmo para sua Máquina Analítica.
  2. Alan Turing: considerado o pai da computação, responsável pela "Enigma", máquina de criptografia, que quebrou o código dos nazistas, dando um importante passo para a vitória dos Aliados na 2ª Guerra Mundial.
  3. Gracie Hopper: Trabalhou no Mark I e Mark II, criou o termo "bug" (que refere-se ao erro no código) e também a linguagem de Programação Flow-Matic, que serviu de base para o COBOL.
  4. Tim Bernies-Lee: Criador da WWW - Word Wide Web.
  5. Marc Andreessen: Criador do primeiro navegador.
  6. Bill Gates: fundador da Microsoft e criador do Windows.
  7. Steve Wosniak: Um dos fundadores da Apple e criador do Apple I e Apple II, primeiro computador da empresa.
  8. Steve Jobs: Um dos fundadores da Apple, responsável pelos produtos inovadores da empresa, entre eles, o Ipod e o Iphone.

A Comunicação Client versus Serves

(Voltar)

Client

Refere-se o cliente, ou seja, o usuário que está consumindo informação na Web e utilizando nossa aplicação.

Serves

Refere-se aos servidores, é o que mantém todas as informações e arquivos referentes às páginas Web em funcionamento na Web. É como se fosse uma “central” ou um datacenter. Pode ser público, como o DNS, que armazena os IP's de todos os sites criados ou pode ser privado como um serviço de hospedagem de sites.

Ferramentas e Conhecimento necessário para a criação de Aplicações Web

(Voltar)

Conhecimento

Para a criação de página Web é necessário o conhecimento de HTML (HiperText Markup Language), CSS (Cascading Style Sheets) e JavaScript. O HTML como diz o próprio nome, é uma linguagem de marcação para criação e estruturação de conteúdo; o CSS é uma linguagem de estilo, ou seja, é utilizada para formatação do conteúdo criado e estruturado com o HTML (seria a parte do design de um site). Por fim, o JavaScript é uma linguagem de programação, utilizado para que o site seja funcional, é o JavaScript que “dá a vida” a uma página web, que permite a interação dos usuários com o site.

Ferramentas

Para começar a criar uma página web é necessário um editor de código, existem vários atualmente, mas recomendo o Visual Studio Code, já que com ele podemos programar em várias linguagens (HTML, CSS, JS, Python, PHP), temos acesso a várias extensões (aumentando a produtividade na hora de programar) e podemos integrar nossos projetos diretamente ao GitHub.

As extensões permitem um melhor desenvolvimento na hora de programar. Para o desenvolvimento web são recomendadas duas extensões: o Emmet (te ajuda a escrever o código com maior velocidade) e o Live Server (permite você visualizar e interagir com sua página web ao mesmo tempo que você está desenvolvendo-a).

Além de um editor de texto devidamente configurado, os navegadores possuem uma ferramenta muito útil - o “Inspetor de Elementos” ou “Ferramenta do Desenvolvedor”, que permite ver o HTML do site aberto no navegador, com todos os detalhes do seu código, incluindo o CSS e o JavaScript.

O HTML na prática

(Voltar)

Estrutura Básica

Ao criar um site na web, é comum (esperado) que ele tenha várias páginas HTML na sua composição. Mas todo site possui uma página principal, aquela que aparece assim que você digita a URL do site na barra de pesquisa do navegador. A esta página principal chama-se “index” (com letras minúsculas mesmo) e é a partir dela que um site é construído.

Tags

Como dito anteriormente, o HTML é uma linguagem de marcação que estrutura elementos em uma página web. Isto é possível graças aos elementos dentro dele, representados pelas Tags. São elas que fazem a real demarcação de um conteúdo dentro do HTML, indicando o seu início e o seu fim. Uma tag é escrita da seguinte forma:

< nome_da_tag > conteúdo < /nome_da_tag >
Nota: os nomes das tags são escritos sempre com letras minúsculas
Todo documento HTML começa com a tag < html > e ele termina com a tag de fechamento < /html> . Além disso, o HTML é dividido em duas partes:
  1. Cabeçalho: dentro dele ficam as informações sobre a página HTML que serão lidas pelo navegador antes dele carregar a página. É delimitado pelas tags < head> e < /head>
  2. Corpo: dentro dele fica todo o conteúdo que aparece ao usuário assim que a página é carregada pelo navegador. É delimitado pelas tags < body> e < /body>
Portanto a estrutura básica de um HTML fica assim:

< html >

< head>
as meta informações sobre a página vão aqui
< /head>

< body>
o conteúdo que aparece para o usuário é criado e estruturado aqui
< /body>

< /html> .

Dentro da tag body vai se adicionando outras tags, estruturando o conteúdo apresentado. Vamos agora conhecer as tags básicas do HTML para a construção de textos na web:

    • Atributos das Tags

    Além das tags para a indicação e organização de elementos no HTML, temos também os atributos, que são as características e informações que um elemento pode ter, permitindo a sua configuração. O atributo pode ser indicado dentro da própria tag do elemento, é possível (e normal) uma tag apresentar mais de um atributo. Os atributos seguem as mesmas convenções das tags e também são escritos e nomeados(quando necessário) com letras minúsculas. Dentre os atributos, podemos destacar:

    id: identifica a tag.

    style: faz a estilização (adicionar cor e/ou um formato diferente) da tag.

    class: identifica um conjunto de tags. É utilizada para padronização.

    type: é um atributo específico da tag input, espefificando qual o tipo de dado (texto, numérico ou cor) será permitido na entrada de dados.

    src: é um atributo específico de aglumas tags como img, audio, video track, etc. Espefificando qual é fonte(source) do conteúdo.

    href: utilizado dentro da tag < a >, que define um hiperlink. O href indica a URL do link.

    title: apresenta iuma informação extra do elemento, seu título. Quando é utilizada dentro da tag head, apresenta o título da página na aba do navegador.

    width / height: configura a largura(width) e a altura(height) do elemento.

    lang: utilizado dentro da tag html, informando o idioma da página web, auxiliando o navegador e também em pesquisas feitas em SEO's.

    alt: é um atributo específico da tag img, utilizado para a acessibilidade, ou quando a imagem não aparece na pagina web, ele fornece um texto com a descrição do conteúdo da imagem.

    Importante: atributo vem acompado do sinal de igualdade, para indicar o valor do atributo. O valor, mesmo numérico, deve ser colocado entre duas aspas.

    Tipografia e Fontes

    (Voltar)

    Outro ponto importante que deve ser considerado é a Tipografia, isto é, os tipos de fontes mais indicadas para uma melhor experiência do usuário em sua página web. Na internet é possível encontrar vários tipos de fontes - o próprio Google possui um site com várias fontes. Mas, além da questão estética, é importante destacar o aspecto do layout responsivo, isto é, sua fonte deve ser legível em vários dispositivos (computador, smartphone ou tablet).

    Na tipografia existem cinco principais famílias, por qual todas as fontes são categorizadas. São elas:
    • Cursiva
    • Fantasia
    • Serifada(Serif)
    • Sem Serif
    • Monoespaço
    Ao escolher uma fonte para sua página Web, você precisa se certificar de fazer o update do arquivo no servidor e verificar se os principais navegadores reconhecem e carregam sua fonte. Por isso, existem as Web Safe Fonts, um conjunto de fontes que já estão préinstaladas como fontes 'padrão' na maioria dos dispositivos e serviços, independente do sistema, garantindo que seu texto será carregado e legível para o usuário. As Web Safe Fonts de maior destaque são:
    • Arial (sans-serif)
    • Verdana (sans-serif)
    • Tahoma (sans-serif)
    • Trebuchet MS (sans-serif)
    • Times New Roman (serif)
    • Georgia (serif)
    • Garamond (serif)
    • Courier New (monospace)
    • Brush Script MT (cursive)

    Para conhecer os tipos de fontes que existem, o Google possui uma página somente para isso, onde é possível esolher e testar sua fonte em vários formatos (letra maiúscula, minúscula, negrito, etc.). Acesse:

    Links

    Como já foi dito, um site não é feito apenas com uma página HTML, mas várias. Portanto, os links são necessários para conectarmos uma página a outra, criando uma navegabilidade pelo site.

    Para conectar uma página html na outra, utilizamos a tag âncora < a href="" > < /a>. Significa que o link está ancorando, indo para algum lugar. Em um texto, a tag fica desta forma:

    < a href="" > < /a>.
    < a href="URL" > tútilo do conteúdo ou página < /a >

    É importante citar que a tag a href aceita o atributo target, com os valores "blank" ou "self", sendo possível decidir como a página web do link irá abrir - em uma nova aba ou na mesma aba. Para uma melhor navegabilidade e para manter o usuário no site, o ideal é escolher o valor "self", abrindo a nova página na mesma aba.

    Para uma melhor navegabilidade dentro da própria página web, quando o usuário está consumindo o conteúdo e "dá um scroll" para baixo, podemos utilizar o método da ancoragem, para facilitar a "volta" do usuário para o início da página. Utilizando o atributo id, nomeie a tag, depois dentro da tag a href, chame o elemento junto com o símbolo #. Exemplo:

    < h2 id="ada-lovelace" > Ada Lovelace, a primeira programadora < /h2 >
    < p > < a href="#ada-lovelace" > Clique aqui para voltar para o título < /a > < / p>

    No exemplo acima temos um título de um texto dentro da tag h2. Considerando que o texto fosse longo, ao chegar ao final da página, o usuário poderia clicar em "Clique aqui para voltar para o título" para retornar ao início do texto automaticamente, não precisando "rolar o scroll", o que deixa a navegabilidade muito melhor.
